Replace toothed belt tensioner

J435210

  Remove

1.   Remove toothed belt
2.   Remove toothed belt tightening roller (1)
•   Unscrew bolt (2)


  Install

3.   Fit toothed belt tightening roll
•   Attach toothed belt tension roller spring
•   Screw in toothed belt tension roller bolt
•   Turn tension roller approx. 180° counter-clockwise
– Tighten bolt
4.   Install toothed belt
