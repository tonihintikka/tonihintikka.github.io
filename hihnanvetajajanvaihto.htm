Replace toothed belt drivewheel

J469600

  Remove

1.   Remove toothed belt
2.   Remove toothed belt drive gear (1)
•   Attach KM-662-C
•   Unscrew bolt
– Counterhold KM-662-C
•   Use KM-161-B (2)
•   Remove plate


  Install

3.   Install toothed belt drive gear
•   Insert plate
•   Insert toothed belt drive gear
– Note disc spring
•   Counterhold KM-662-C
•   Tighten bolt 196 Nm
4.   Install toothed belt
